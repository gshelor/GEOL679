Q4. This dataset was taken from a cruise where the instruments recorded navigation and magnetic data in two separate files. We can align the timing of the two files through interpolation using the workflow below. When this is finished, we will have added magnetic values to the navigation DataFrame.

  • First, use interp1d(...) to fit a interpolant to the 'timediff' and 'magnetics' columns of the magnetics DataFrame.
  • Use this fit to estimate the magnetics values at the times listed in the navigation DataFrame. Add these values as a new column to navigation DataFrame.
  • Display the final results using head()
In[]:
## fitting interpolant to timediff and magnetics columns
fit = interp1d(mag_df['timediff'], mag_df['magnetics'], kind = 'linear')

## using fit to add new column of estimated magnetics values to nav_df
nav_df['magnetics'] = fit(nav_df['timediff'])

Q10. Use griddata(...) to grid the points using the 'cubic' method.

Plot the resulting grid using plt.contourf(...), label x and y axes, and add a colorbar with the colormap of your choice. Set the x and y tick values manually to avoid overlapping labels.

grid_cubic = griddata(points, arr3, (grid_x, grid_y), method='cubic')
